CTF 25 — Import & Export

Common Transfer File · DfE statutory pupil data format

What is CTF? The Common Transfer File is the DfE-mandated XML format used by every state school MIS (SIMS, Arbor, Bromcom, ScholarPack, Integris, etc.) to transfer pupil data when children move schools. Exporting CTF is a statutory requirement under the Education (Pupil Information) (England) Regulations 2005.
