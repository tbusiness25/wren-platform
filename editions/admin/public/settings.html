🔒 Access Control
Allow full portal access from outside school network When OFF, staff accessing from outside the school Tailscale network can only reach the HR portal — no child or learning data. When ON, full access is allowed remotely.
✓ On Tailscale (100.x.x.x) — always full access ✕ Remote — HR only
How this works: Wren checks the requesting IP address. Connections from the 100.x.x.x Tailscale network always have full access. Connections from any other IP are either allowed (toggle ON) or restricted to HR only (toggle OFF). Recommended: keep OFF for safeguarding — staff should use Tailscale for remote access to child data.

🔒 Backup & Recovery
📖 Recovery Guide
Step-by-step recovery
  1. SSH into the Wren server: ssh toby@your-server
  2. Run: bash /app/scripts/recover.sh
  3. Confirm when prompted — the script lists available snapshots and restores the latest
  4. Containers restart automatically after restore
  5. If database is unrecoverable, restore from USB drive: rsync -a /media/wren-backup/ /app-backups/
  6. Contact Wren support if USB is also unavailable
