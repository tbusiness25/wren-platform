🏖️ Away Mode
Away mode When ON, Aria greets callers with "The manager is away until [return date]" and offers cover if a deputy is available. Fires an email OOO webhook to N8N.
Note: The manager switches this ON when they leave. The N8N flow handles the Google Workspace OOO responder. Toggle OFF to revert everything. If the cover person is unavailable on a given day, Aria will say so automatically.

⚠ Danger zone — private instance

Permanently delete every child, observation, message and log from this instance (staff logins, settings and frameworks are kept), then sign you out. Use it to clear personal data for GDPR after you've exported your reports. This cannot be undone.
